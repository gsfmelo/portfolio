Projeto de Bicicletaria

O projeto foi realizado como trabalho final da disciplina de Lógica de Programação no curso de Análise e Desenvolvimento de Sistemas. O objetivo do projeto é ser uma bicicletaria é que faz cadastro de peças de bicicleta, informando seu valor, nome e fabricante. Ao inserir esse dados, é gerado um código único para cada peça. É possível consultar suas peças e removê-las também. O projeto é executado através do PyCharm de acordo com o input desejado. Clique no link abaixo para acessá-lo.
